### 构造器属性

> 当我们创建对象时，实际上同时也赋予了该对象一种特殊的属性---即构造器属性。该属性实际上就是一个指向用于创建该对象的构造器函数的引用。

``` javascript
function Hero(a) {
	this.a = a;	
}
var h = new Hero("mm");
alert(h.constructor); // function Hero(a) {this.a = a}
```
